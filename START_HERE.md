## 🚀 빠른 시작 가이드

### 1단계: 패키지 설치

```bash
npm install
```

### 2단계: 개발 서버 실행

```bash
npm run dev
```

브라우저에서 **http://localhost:3000** 접속하세요!

- 영어, 일본어, 중국어(간체/번체), 한국어, 스페인어, 프랑스어, 독일어, 러시아어, 아랍어, 태국어, 베트남어

1. **Hero** - 히어로 + 언어 선택기

5. **Process** - 5단계 워크플로우
